# Spirale d'Ulam v1

Interactive Ulam spiral visualizer for prime number families up to 1 million.

## Features

- Renders 1,000,000 integers on the Ulam spiral with real-time pan/zoom
- 8 prime families color-coded: Prime, Sophie Germain, Mersenne, Fermat, Heegner, Twin, Palindrome, Safe
- Rope drawing tool: connect any two primes and classify the chord
- 7 rope families: Cordier, Antipode, Diagonal, Orthogonal, Résonant, Frère, Écho
- Info cards on tap with gap, length, ratio R, slope, angle
- Touch & mouse support (pinch-zoom, drag, double-tap zoom)
- Progressive loader with step feedback

## Tech Stack

- Pure HTML5 Canvas 2D / CSS3 / Vanilla JavaScript
- No external libraries, no WebGL, no frameworks
- Eratosthenes sieve (domain-public algorithm)
